## GitHub launch blurb

RanchMind is a closed-loop agent runtime for people who want more than a single chat window.

- **Lobster** handles execution and coding backends
- **Horse** handles scheduling, messaging, and dispatch
- **Human** handles memory, review, and operator control

Instead of choosing between agent UX, background automation, and CLI power, RanchMind treats all three as first-class planes in one architecture.
